1. В королевстве роботов необходимо посчитать количество роботов, которые готовы к работе. Каждый робот отвечает "да" или "нет". Используйте цикл while для запроса у пользователя ответа для каждого робота.

Инструкция:
1. Создайте переменную для хранения количества готовых роботов.
2. Используйте цикл while, чтобы продолжать спрашивать у пользователя, готов ли следующий робот.
3. Для каждого ответа "да", увеличивайте счетчик готовых роботов.
4. Остановите цикл, когда пользователь введет "стоп".
5. Выведите общее количество готовых роботов с помощью alert.


2. Задача: Напишите программу, которая использует цикл while и условие для классификации роботов по состоянию: "готовы", "нуждаются в ремонте" или "списаны". Используйте prompt и alert.

Инструкция:
1. Создайте переменные для хранения количества роботов в каждом состоянии: ready, needsRepair, decommissioned.
2. Используйте цикл while, который будет продолжаться до тех пор, пока пользователь не введет "стоп".
3. Внутри цикла используйте prompt, чтобы запросить состояние робота.
4. Увеличивайте соответствующий счетчик в зависимости от ввода пользователя.
5. После завершения цикла выведите результаты с помощью alert.


3. В королевстве роботов проводится инвентаризация. Напишите функцию, которая будет собирать информацию о каждом роботе, используя цикл while, условия, и окна prompt, alert, и confirm.

Инструкция:
1. Создайте функцию inventoryRobots.
2. Внутри функции используйте переменные для хранения количества готовых, нуждающихся в ремонте и списанных роботов.
3. Используйте цикл while для запроса состояния каждого робота.
4. В зависимости от ответа пользователя, увеличивайте соответствующий счетчик.
5. Используйте confirm для подтверждения окончания ввода данных.
6. Выведите результаты с помощью alert.
